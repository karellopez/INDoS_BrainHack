# INDoS BrainHack Workshops

Static landing page for the INDoS BrainHack workshops on:

- Multimodal raw-to-BIDS curation with BIDS Manager
- Reproducible MEG/EEG quality control with MEEGqc

The page is intentionally dependency-free. It can be served directly from GitHub Pages or opened locally as `index.html`.

## Local Preview

Open `index.html` in a browser, or run a simple static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
